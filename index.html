Welcome to Palindrome Checker

A palindrome is a word, number, or other sequence of characters which reads the same backward as forward, such as "madam" or "racecar".

About this challenge

The palindrome checker allows a user to input a string and return whether or not the inputted string is a palindrome.

  • Takes in a user string.
  • Reverses the user's string.
  • Checks the reversed string against the original string.
  • Returns whether or not the inputted string is or is not a palindrome.

        function palindrome() {
        let userStr = document.getElementById("str").value.toLowerCase().replace(/[`~!@#$%^&*()_|+\-=?;:'",.<>\{\}\[\]\\\/]|\s/gi, '');
        let rev = userStr.split('').reverse().join('');
        if (userStr == rev) {
        let resultp = userStr + " is a palindrome";
        document.getElementById("resultp").innerHTML = `${resultp}`;
        }
        else {
        let resultp = userStr + " is not a palindrome";
        document.getElementById("resultp").innerHTML = `${resultp}`;
        }
        }
